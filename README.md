# Visitor Pass Management System (MERN)

Simple project for managing visitor passes. Made by someone with only 6 months experience in web dev. Basic features implemented.

## Features
- Login Register
- Visitor Registration
- Dashboard
- QR Code Pass
- Basic check in

## Setup Instructions

Backend:
cd backend
npm install
npm run dev  (or node server.js)

Frontend:
cd frontend
npm install
npm run dev

Create .env in backend with MONGO_URI

Run seed.js for demo data: node seed.js

For full, install MongoDB.
